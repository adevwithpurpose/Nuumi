# Nuumipet Launch Readiness Checklist

## Theme And Preview

- [ ] `shopify theme check` passes from repo root
- [ ] all five product templates exist and reference valid sections
- [ ] preview renders for all five products
- [ ] shared PDP stack works without review or subscription apps installed

## Product Records

- [ ] all five Shopify products exist
- [ ] all five handles match `docs/product-rollout-matrix.md`
- [ ] template suffixes match intended targets
- [ ] draft vs active product status is intentional

## Content And Editability

- [ ] all five local product docs are complete
- [ ] product-specific copy is populated in templates and/or metafields
- [ ] no empty FAQ, trust, or guarantee sections render
- [ ] Theme Editor users can update major PDP content without editing code

## Launch Gates

- [ ] homepage product links point to the correct products
- [ ] policy pages exist or are explicitly marked merchant-follow-up
- [ ] media inventory is present or missing items are documented
- [ ] sales channel visibility is known
- [ ] payment/test-mode review is completed or explicitly deferred to merchant operations
