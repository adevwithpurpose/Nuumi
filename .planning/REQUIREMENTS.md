## Theme & Workflow

- [ ] **THEME-01**: All sections must be built as reusable Shopify sections.
- [ ] **THEME-02**: All content (text, images, etc.) must be editable through the Shopify Theme Editor using settings and dynamic sources (metafields).
- [ ] **THEME-03**: The theme must be based on Shopify's Dawn theme.
- [ ] **THEME-04**: The project must use the Shopify CLI for local development.
- [ ] **THEME-05**: The theme must be connected to a GitHub repository for version control.
