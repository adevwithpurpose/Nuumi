<task type="checkpoint:human-action" gate="blocking">
  <name>task 3: Create and Connect GitHub Repository</name>
  <decision>A remote repository needs to be created on GitHub and connected to the local project. Shopify's GitHub integration also needs to be configured.</decision>
  <context>
OpenCode cannot create repositories on GitHub or configure Shopify App integrations on your behalf. These steps require access to your personal GitHub account and Shopify admin panel.
</context>
  <how-to-action>
1.  **Create a new, empty repository on GitHub.com.** Do NOT initialize it with a README or .gitignore file.
2.  **Copy the repository URL** (e.g., `https://github.com/your-username/nuumipet-theme.git`).
3.  **Connect the local repository to the remote:**
    ```bash
    # Run these commands inside the `nuumipet-v1` directory
    git remote add origin YOUR_COPIED_URL
    git branch -M main
    git push -u origin main
    ```
4.  **Connect the theme in Shopify Admin:**
    - Go to your Shopify Admin > Online Store > Themes.
    - Click "Add theme" > "Connect from GitHub".
    - Follow the prompts to log in to GitHub and select the new repository and the `main` branch.
</how-to-action>
  <resume-signal>Type "approved" once you have created the GitHub repository, pushed the local code, and connected the theme in the Shopify admin.</resume-signal>
