# Nuumipet Shopify Storefront

## Project Overview

Nuumipet is a new Shopify storefront for a lean pet-health e-commerce brand. The project aims to create a premium, clean, and high-converting store that feels trustworthy and modern. The theme will be based on Shopify's Dawn theme but heavily customized to meet the brand's specific needs. A key requirement is that all essential content should be editable through the Shopify Theme Editor, allowing the client to manage the store without developer assistance.

## Core Value

The core value of this project is to create a Shopify theme that is both visually stunning and easy for the client to manage. The theme should reflect a premium and trustworthy brand identity while providing a seamless and intuitive user experience.

## Scope

The initial scope of the project includes:
- A custom homepage built from scratch.
- A reusable Product Detail Page (PDP) framework.

## Design Direction

- **Base Theme:** Shopify Dawn (heavily customized)
- **Design Inspiration:** PetLab for authority and trust, but with a cleaner, calmer, and more premium aesthetic.
- **Brand Feel:** Blue-led, clinical-but-friendly, mobile-first, with strong visual polish.
- **Store Model:** A lean storefront featuring five products, all accessible from the homepage.
